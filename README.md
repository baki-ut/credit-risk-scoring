# Credit Risk Scoring 🏦

## О проекте
Пет-проект по построению системы кредитного скоринга (предсказание вероятности дефолта заемщика). 
Проект разрабатывается с упором на современные стандарты ML-инженерии (MLOps) и чистый код.

## Бизнес-задача
Определить клиентов, которые с высокой вероятностью уйдут в серьезную просрочку (90+ дней). Это позволит банку заранее отказывать в кредите высокорискованным заемщикам или предлагать им другие условия, минимизируя финансовые потери.

## Стек технологий 🛠
* **Управление проектом:** `uv` (ультрабыстрый менеджер зависимостей), `Ruff` (линтер/форматтер).
* **Машинное обучение и данные:** `pandas`, `numpy`, `scikit-learn`, `catboost`.
* **Визуализация:** `matplotlib`, `seaborn`.
* **API:** `FastApi`.
